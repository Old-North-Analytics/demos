# demos

Interactive tools used as live portfolio demonstrations on the ONA website.
Each subfolder is a self-contained single-page application.

| Demo | URL |
|---|---|
| Crew Scheduler | DispatchIQ; automated crew assignment with constraint matching and AI-assisted routing (field ops) |
| Rent Management | End-to-end rent tracking and tenant management; replaces manual spreadsheet workflows |
| Retail Chatbot | Retail intelligence dashboard with AI-powered plain-English query interface for sales and inventory KPIs |
| Field Scheduling | Field services scheduling dashboard (landscaping/field ops context with weekly calendars) |
| Job Profitability | Margin tracking by job; estimated vs. actual, regression-based scenario modeling |
